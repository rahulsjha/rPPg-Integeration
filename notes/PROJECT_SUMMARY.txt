Near Real-Time rPPG Integration - Project Summary
Date: 2026-05-05

1) Which model we used
- Primary integrated model: Open-rppg backend using rppg.Model('FacePhys.rlap').
- Source: https://github.com/KegangWangCCNU/open-rppg
- CV front-end in pipeline: OpenCV Haar Cascade face detector + forehead ROI extraction.
- Chunk logic: 5-second windows, per-window inference, confidence-weighted BPM aggregation.

2) What the assignment asks
- Input a 60-second face video.
- Process incrementally in 5-second chunks.
- Produce chunk BPM and final BPM.
- Include runtime/performance metrics.
- Bonus points for Respiratory Rate and additional biomarkers.

3) How we solved it
- Installed and integrated open-rppg.
- Added an Open-rppg adapter in src/open_rppg_backend.py.
- Updated run_chunked_prototype.py to support model-backend=open-rppg.
- Each 5-second chunk:
  - detect/crop face ROI frames,
  - run open-rppg on face tensor,
  - store chunk BPM and confidence.
- Final BPM uses robust confidence-weighted averaging.
- RR uses model HRV breathing rate when available, with fallback RR estimator from full ROI signal.

4) Current result explanation (open-rppg run)
Input: input/assignment_60s.mp4

- Chunk-level BPM generated for all 12 windows.
- Final BPM: 103.0
- Respiratory rate: 15.0 brpm
- Runtime metrics:
  - wall_time_sec: 44.866
  - avg_chunk_compute_ms: 701.735
  - p95_chunk_compute_ms: 887.348
  - effective_pipeline_fps: 33.477
  - realtime_factor_x: 1.339

Interpretation:
- Throughput is faster than real-time overall (RTF > 1).
- Open-rppg deep-model chunk inference increases per-chunk latency versus lightweight signal-only methods.
- Chunk BPM variability still exists under short-window constraints.

5) How accuracy is measured
With reference HR (time_sec,bpm), evaluator reports:
- MAE, RMSE, MAPE
- within ±3/±5/±10 bpm
- Pearson correlation
- overall BPM absolute error

Without reference HR, evaluator reports proxy quality:
- median confidence
- BPM std / IQR
- outlier chunk ratio

6) Test commands (plain text)
Install model package:
python3 -m pip install open-rppg

Run chunked Open-rppg inference:
python3 run_chunked_prototype.py --video input/assignment_60s.mp4 --chunk-sec 5 --model-backend open-rppg --open-rppg-model FacePhys.rlap --json-out notes/chunked_rppg_output.json

Run no-ground-truth evaluation:
python3 evaluate_accuracy.py --pred notes/chunked_rppg_output.json --out notes/accuracy_report.json

Run evaluation with reference HR + RR:
cp notes/reference_hr_template.csv notes/reference_hr.csv
python3 evaluate_accuracy.py --pred notes/chunked_rppg_output.json --ref-bpm-csv notes/reference_hr.csv --ref-rr 14.8 --out notes/accuracy_report_with_gt.json
